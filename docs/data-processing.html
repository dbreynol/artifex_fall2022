2.1 Cleaning up strings

Some aspects of this dataset that are distracting for analysis include:

  1. The meaningless (to us) numbers that precede the customer name and business unit.
  2. The meaningless letters that precede the product description.
  3. The product variable includes the size which would ideally exist in its own column (i.e., we want a size column whose first entry is 9)

Let’s work through each of these using the functions str_sub, str_locate, and str_split. Let’s first go over what these functions do and how they can be used for this task.

str_sub("The big short", start = 5, end = 7) # big

# the input can also be a vector
x <- c("Apple", "Banana", "Pear") # character vector
str_sub(x, 1, 3) # "App" "Ban" "Pea"
str_locate("This", "s") # gives the starting and ending position of the 's' in 'This'

The patterns that are located can be much more general than a literal letter. These patters are known as regular expressions. For example, suppose we would like to locate the part of the string that consists of the letter a followed by any number. The regular expression that represents any digit is \d.

str_locate("aardvark a3", "a\\d") # 10 11

For more on matching regular expressions in R, see this document on the topic.
